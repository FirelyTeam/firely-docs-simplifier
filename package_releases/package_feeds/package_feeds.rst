Private package releases with feeds
===================================

.. important::

   Package feeds are available from the Professional plan and up. `See the pricing page for details. <https://simplifier.net/pricing>`_

Package feeds let you publish and consume private FHIR packages within Simplifier, with controlled access. Start with the getting started guide, then use the technical reference for the details.
